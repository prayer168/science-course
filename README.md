# 115 學年度國小國中自然三版本課程架構

這個網頁整理康軒、翰林、南一 115 學年度國小三至六年級自然科，以及國中七至九年級自然科學與理化課程架構，提供年級篩選、版本篩選與關鍵字搜尋。

## 使用方式

直接開啟 `index.html`，或透過 GitHub Pages 瀏覽：

https://prayer168.github.io/science-course/

## 內容範圍

- 國小三、四年級：植物、動物、空氣與水、磁力、天氣、聲光、電路、昆蟲、地表環境與自然資源。
- 國小五、六年級：水溶液、力與運動、燃燒與生鏽、熱、電磁、天氣變化、地表作用、簡單機械與生態永續。
- 七年級：生命科學、生物分類、生殖遺傳、生態與環境。
- 八年級：基本測量、物質、波動、光、熱、原子分子、化學反應、酸鹼鹽、氧化還原與理化銜接。
- 九年級：運動與力、功與能、電學、電磁、地球科學、天文與永續環境。
